# susaf-programming-tool
1. Please first ensure that there is a file directory named "assignment" on the E disk.（"E:\\assignment"）
2. The front-end code is deployed on the local port 8080, while the back-end code is deployed on the local port 8081.
3. Use "npm install" to install module when deploy frontend code then use "npm run serve" to start.

Sequence for usage

1. Open the system
2. Upload the assignment code (.zip)
3. Review the feedback
4. Complete the questionnaire
5. Get a visualization about the sustainable awareness 
6. Revise the code and upload again
7. Complete the questionnaire
8. Get a visualization
9. effect chain can let you have a clear understanding of the link from code issues to sustainability development goals
format like "infinite loop -> Cpu runs forever -> resource wasting -> SDG 7 Affordable and Clear Energy"
